系列: Windows 存储驱动开发实战
适用对象: 嵌入式驱动开发者、Windows 内核工程师、存储系统研究人员

系列简介

本系列文章详细介绍 Bayhub SD Host Controller 驱动在 Windows Storport 框架下的开发与调试实践，涵盖：

  • DMA Remapping/IOMMU 兼容性处理
  • PIO/DMA 传输模式设计与实现
  • Win10/Win11 Storport 框架差异分析
  • 多线程到单线程架构迁移
  • 内核调试工具与实战技巧

核心技术点

技术领域 涉及内容
驱动框架 Storport Miniport, WDF, ScsiPort
DMA 技术 ADMA2/ADMA3, SDMA, IOVA/IOMMU, DMA Remapping
传输模式 PIO 模式, SDMA 直接路径, 混合模式
系统集成 Kernel DMA Protection, PCIe, ACPI/DMAR
调试技术 WinDbg, ETW, DIFx 日志, Process Monitor

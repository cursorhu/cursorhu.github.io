1. Linux MMC жЎҶжһ¶зҺ°зҠ¶

Linux MMC driverжҳҜж”ҜжҢҒеҢ…жӢ¬SDеҚЎпјҢeMMCеҚЎзӯүзӯүпјҢеұһдәҺMultiMediaCardи®ҫеӨҮе’ҢжҺҘеҸЈзҡ„й©ұеҠЁ
е…¶жәҗз Ғи·Ҝеҫ„дҪҚдәҺKernel source codeзҡ„drivers/mmcи·Ҝеҫ„, еӨҙж–Үд»¶дҪҚдәҺinclude/linux/mmc
mmcжәҗз ҒеҲҶдёәcore/hostдёӨеұӮпјҢжҳҜдёәдәҶи§ЈиҖҰпјҡ

  • йҖҡз”Ёзҡ„SD/eMMCжөҒзЁӢ(core)
  • е…·дҪ“зҡ„зЎ¬д»¶ж“ҚдҪңжөҒзЁӢ(host)пјҢеңЁжӯӨеұӮеҸҲеҸҜеҲҶдёәйҖҡз”Ёзҡ„SDHCIжЎҶжһ¶е’ҢйқһSDHCIжЎҶжһ¶пјҢеҗ„eMMC/SD hostеҺӮе•Ҷе®һзҺ°жңҖеә•еұӮdriverж—¶пјҢеҸҜд»ҘйҒөеҫӘSDHCIжЎҶжһ¶дёӢзҡ„API, й—ҙжҺҘе®һзҺ°coreеұӮе®ҡд№үзҡ„ж–№жі•(driverз§°дёәoperations), д№ҹеҸҜд»ҘдёҚйҒөеҫӘSDHCIжЎҶжһ¶пјҢзӣҙжҺҘе®һзҺ°coreеұӮе®ҡд№үзҡ„ж–№жі•гҖӮ

жң¬ж–ҮйҮҚзӮ№е…іжіЁmmcжЎҶжһ¶еҜ№SDеҚЎй©ұеҠЁзҡ„ж”ҜжҢҒ

  • SD Host Controller Specification Ver7.0 (д»ҺhostжҺ§еҲ¶еҷЁи§’еәҰпјҢжҸҸиҝ°SD 7.0, SD 4.0, д»ҘеҸҠжӣҙж—©зүҲжң¬SDзҡ„и§„иҢғ)

1.2 Linux MMCжЎҶжһ¶еҜ№SDеҚЎзҡ„ж”ҜжҢҒ

еҹәжң¬жҰӮеҝөпјҡеҸӘжңүmmcжЎҶжһ¶зҡ„coreеұӮж”ҜжҢҒжҹҗз§ҚSDжЁЎејҸпјҢhostеұӮжүҚиғҪе®һзҺ°иҝҷз§ҚжЁЎејҸпјӣеҰӮжһңcoreеұӮйғҪдёҚж”ҜжҢҒпјҢеҸӘиғҪеҺӮе•ҶиҮӘе·ұејҖеҸ‘coreеұӮпјҢд»ҘpatchиЎҘдёҒзҡ„ж–№ејҸеҸ‘еёғгҖӮ

coreеұӮеҜ№дәҺдёҠиҝ°дёүз§ҚSDжЁЎејҸзҡ„ж”ҜжҢҒпјҡ

  • Linux kernel 5.11 д»ҘеүҚпјҢеҸӘж”ҜжҢҒUHS-IеҸҠе…¶жӣҙдҪҺйҖҹеәҰзҡ„legacy-SDжЁЎејҸ
  • Linux kernel 5.11 ејҖе§ӢпјҢеңЁcoreеұӮж·»еҠ дәҶSD expressзҡ„ж”ҜжҢҒ
  • зӣ®еүҚжІЎжңүUHS-IIзҡ„ж”ҜжҢҒпјҢеҸӘжңүжҸҗдәӨеҫ…е®Ўж ёзҡ„пјҢеҸӮиҖғпјҡlore.kernel.org/Jason Lai/patch

hostеұӮеҜ№дәҺдёҠиҝ°дёүз§ҚSDжЁЎејҸзҡ„ж”ҜжҢҒпјҡ

  • UHS-I: еҹәжң¬hostзӣ®еҪ•зҡ„еӨ§еӨҡж•°SDеҺӮе•Ҷй©ұеҠЁйғҪж”ҜжҢҒпјҢеҫҲеӨҡз¬ҰеҗҲsdhciжЎҶжһ¶
  • SD express: RealtekеҹәдәҺLinux kernel 5.11зҡ„coreеұӮAPI, е®һзҺ°дәҶ й©ұеҠЁзҡ„hostеә•еұӮйғЁеҲҶпјҢеҸӮиҖғkernelзҡ„host/rtsx_pci_sdmmc.c, е…¶жІЎжңүдҪҝз”ЁSDHCIжЎҶжһ¶гҖӮ
  • UHS-II: еҸӘжңүд»Ҙpatchж–№ејҸе®һзҺ°зҡ„пјҢеҸӮиҖғ# linux-uhs2-gl9755пјҢе…¶е®һзҺ°дәҶcore/host-sdhci/host vendorеӨҡдёӘеұӮж¬Ўзҡ„UHS-IIж”ҜжҢҒгҖӮ

з»јдёҠжүҖиҝ°пјҢжң¬ж–ҮеҸӮиҖғuhs2-gl8755 patchпјҢе®һзҺ°иҮӘе·ұзҡ„SD UHSII driverгҖӮ

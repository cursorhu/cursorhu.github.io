е®үиЈ…дёӯж–Үж”ҜжҢҒпјҡ

и®ҫзҪ®зі»з»ҹиҜӯиЁҖдёәдёӯж–Үпјҡ

snapеҢ…з®ЎзҗҶе·Ҙе…·

SnapжҳҜCanonicalејҖеҸ‘зҡ„LinuxеҢ…з®ЎзҗҶе’ҢиҪҜд»¶йғЁзҪІе·Ҙе…·гҖӮ

е®үиЈ…е’ҢдҪҝз”ЁеҸӮиҖғ How to Install Snap on Ubuntu

зү№зӮ№пјҡдё°еҜҢзҡ„з¬¬дёүж–№е·Ҙе…·еә“пјҢеҢ…жӢ¬ејҖжәҗе·Ҙе…·е’Ңй—ӯжәҗе·Ҙе…·пјӣдәҢиҝӣеҲ¶е®үиЈ…пјҢдёҚжҳҜжәҗз Ғзј–иҜ‘

зӣёжҜ”aptпјҢе…¶жҹҘжүҫе·Ҙе…·е’Ңе®үиЈ…жһҒдёәз®ҖеҚ•пјҡ

жҹҘзңӢе’ҢеҚёиҪҪsnapе®үиЈ…зҡ„еҢ…пјҡ

зӨәдҫӢпјҡе®үиЈ…VSCodeе’ҢChrome

и®ҫзҪ®еҝ«жҚ·й”®

setting -> keyboard -> shortcuts -> custom shortcut -> дёәеә”з”ЁзЁӢеәҸж·»еҠ еҝ«жҚ·й”®

д»ҘжҲӘеӣҫе·Ҙе…·flameshotдёәдҫӢпјҢи®ҫзҪ®еҝ«жҚ·й”®зҡ„commandдёәи°ғз”Ёflameshotзҡ„е‘Ҫд»ӨпјҢжҲӘеӣҫй»ҳи®ӨдҝқеӯҳеҲ°~/Pictures

иҰҒй…ҚзҪ®е…¶д»–flameshotе‘Ҫд»Өзҡ„еҝ«жҚ·й”®пјҢз”Ё man flameshot жҹҘзңӢпјҢеҸӮиҖғ Keyboard shortcuts for Flameshot

TimeshiftеӨҮд»Ҫзі»з»ҹ

22.04зі»з»ҹдјјд№ҺжҜ”иҫғе®№жҳ“жҢӮпјҢиҝӣдёҚдәҶзі»з»ҹжҳҫзӨәвҖқOh noвҖҰ system canвҖҷt recoverвҖҰвҖқпјҢжҜ”еҰӮпјҡ

Nvdiaй©ұеҠЁйҖүжӢ©ејҖжәҗзүҲжң¬xserverе°ұжҢӮдәҶдёҖж¬Ў, recoveryжЁЎејҸзңӢ/var/log/messageжңүnouveauе’Ңnvidia moduleзӣёе…ій—®йўҳ

еӯҰд№ xv6ж—¶е®үиЈ…зј–иҜ‘зҺҜеўғж—¶д№ҹжҢӮдәҶдёҖж¬Ў(дёҚиғҪе®үиЈ…еҲ°/usr/localпјҢеә”иҜҘе®үиЈ…еҲ°/home)пјҢrecoveryжЁЎејҸdpkg reportжҳҫзӨәfailure logпјҡ

йғҪжҳҜжүҫйҒҚеҠһжі•йғҪдҝ®еӨҚдёҚдәҶпјҢеҸӘиғҪйҮҚиЈ…вҖҰ

дёәдәҶи§ЈеҶіжӯӨй—®йўҳпјҢдҪҝз”ЁTimeshiftе°Ҷзі»з»ҹеӨҮд»ҪпјҢеҸӮиҖғ: How to Backup and Restore Linux System Settings With Timeshift

е®үиЈ…timeshiftпјҡ

еӨҮд»Ҫж•ҙдёӘзі»з»ҹпјҢеҢ…жӢ¬/rootе’Ң/home/userпјҢи®ҫзҪ®е®ҡж—¶еӨҮд»Ҫ

еҰӮдҪ•жҒўеӨҚпјҡ

жғ…жҷҜдёҖпјҡзі»з»ҹж— жі•иҝӣе…ҘжЎҢйқўпјҢдҪҶжҳҜеҸҜд»Ҙиҝӣе…ҘrecoveryжЁЎејҸrootж“ҚдҪңпјҡ

еҰӮдёӢеӣҫпјҢз”Ёtimeshift --helpжҹҘзңӢеҗ„з§Қе‘Ҫд»ӨпјҢдҪҝз”Ёtimeshift --restoreжҒўеӨҚжҢҮе®ҡsnapshot

жғ…жҷҜдәҢпјҡзі»з»ҹж— жі•иҝӣе…ҘrecoveryжЁЎејҸпјҢдҪҶжҳҜеӨҮд»Ҫзҡ„snapshotж•°жҚ®иҝҳеңЁ

дҪҝз”ЁUbuntu Live USB пјҢеҚіиЈ…зі»з»ҹзҡ„USBиҝӣе…Ҙtry ubuntuзҺҜеўғпјҢиҒ”зҪ‘жҚўеӣҪеҶ…жәҗе®үиЈ…timeshiftпјҢеҶҚжҒўеӨҚзі»з»ҹзӣҳдёӯзҡ„snapshotж•°жҚ®

жғ…жҷҜдёүпјҡзЈҒзӣҳдёӯзҡ„snapshotж•°жҚ®жҚҹе®іпјҡеҸӘиғҪйҮҚиЈ…зі»з»ҹпјҢдёәдәҶйҒҝе…ҚжӯӨжғ…еҶөеҸ‘з”ҹпјҢеә”иҜҘе°Ҷзі»з»ҹеӨҮд»ҪеҲ°е…¶д»–зЎ¬зӣҳиҖҢдёҚд»…д»…еңЁеҪ“еүҚзі»з»ҹзӣҳ

Clonezillaе…ӢйҡҶзі»з»ҹ

зұ»дјјwindows ghostзҡ„ж•ҙзӣҳе…ӢйҡҶпјҡ

https://www.linuxbabe.com/backup/how-to-use-clonezilla-live

иҮіе°‘йңҖиҰҒдёүдёӘзӣҳпјҡ

еңЁUзӣҳеҶҷе…ҘClonezillaзҡ„live usb isoз”ҹжҲҗClonezilla live USBпјҢеҶҚд»ҘClonezilla live USBеҗҜеҠЁпјҢеҜ№еҫ…еӨҮд»Ҫзҡ„SSDзӣҳеҒҡзі»з»ҹеӨҮд»ҪпјҢеҲ°еҸҰдёҖдёӘSSDжҲ–иҖ…еӨ§USBзӣҳпјӣ

жҒўеӨҚд№ҹжҳҜйңҖиҰҒClonezilla live USB + жңүзі»з»ҹеӨҮд»Ҫзҡ„зӣҳ + зӣ®ж ҮеҶҷе…Ҙзӣҳ

е…ідәҺзі»з»ҹзӣ®еҪ•

/usrпјҡзі»з»ҹзә§зҡ„зӣ®еҪ•пјҢеҸҜд»ҘзҗҶи§ЈдёәC:/Windows/пјҢaptе®үиЈ…зҡ„дёҖиҲ¬еңЁ/usr/binе’Ң/usr/lib

/usr/libпјҡзҗҶи§ЈдёәC:/Windows/System32

/usr/localпјҡз”ЁжҲ·зә§зҡ„зЁӢеәҸзӣ®еҪ•пјҢеҸҜд»ҘзҗҶи§ЈдёәC:/Progrem Files/пјҢз”ЁжҲ·иҮӘе·ұзј–иҜ‘зҡ„иҪҜд»¶й»ҳи®Өе®үиЈ…еҲ°иҝҷдёӘзӣ®еҪ•дёӢ

/optжҳҜз”ЁжҲ·зә§зҡ„зӣ®еҪ•з”ЁжқҘе®үиЈ…еӨ§еһӢзҡ„з¬¬дёүж–№йҷ„еҠ иҪҜд»¶еҢ…пјҢеҸҜд»ҘзҗҶи§ЈдёәD:/Software

ејҖеҸ‘иҝҮзЁӢдёӯдёәдәҶйҒҝе…ҚlibеҶІзӘҒпјҢиҮӘе·ұзј–иҜ‘зҡ„еҢ…е»әи®®ж”ҫеңЁ/home/<е…·дҪ“зҡ„йЎ№зӣ®зӣ®еҪ•>пјҢжӯӨеӨ–жіЁж„ҸиҮӘе·ұзј–иҜ‘еҹәзЎҖеә“и®ҫзҪ®зҡ„LD_LIBRARY_PATHйҖ жҲҗзі»з»ҹеә“й“ҫжҺҘеҶІзӘҒ

дёҠдёӢеҲҶеұҸпјҡдёҚж”ҜжҢҒпјҢе»әи®®дҪҝз”ЁеӨҡжЎҢйқў

жҲӘеӣҫзұ»й…ҚзҪ®

жҲӘеӣҫе·Ҙе…·е’ҢеұҸе№•зј©ж”ҫ

1.жҲӘеӣҫе·Ҙе…·пјҡ

ubuntuй»ҳи®ӨејҖеҗҜдәҶwaylandпјҢдјҡйҳ»зўҚз¬¬дёүж–№жҲӘеӣҫе·Ҙе…·зҡ„еҠҹиғҪгҖӮйңҖиҰҒе…Ҳе…ій—ӯwaylandпјҢдҪҝз”ЁxorgжЎҢйқўпјҡ

жҠҠ #WaylandEnable=false зҡ„#жіЁйҮҠеҺ»жҺүпјҢйҮҚеҗҜз”ҹж•ҲгҖӮеҶҚе®үиЈ…flameshotзӯүжҲӘеӣҫиҪҜд»¶пјҢиғҪжӯЈеёёдҪҝз”ЁеҠҹиғҪгҖӮ

2.еұҸе№•зј©ж”ҫпјҡ

setting -> displays -> Fractional Scalingжү“ејҖпјҢеҚіеҸҜж”ҜжҢҒ125%зј©ж”ҫгҖӮ

еұҸе№•зј©ж”ҫдјҡеј•иө·flameshotжҲӘеӣҫй—ӘзғҒй—®йўҳпјҢи§ЈеҶіж–№жЎҲпјҡдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨеҗҜеҠЁflameshot

issueеҸӮиҖғпјҡFails when fractional scaling <> 100% #564

3.жҲӘеӣҫе·Ҙе…·еҠ еҲ°еҝ«жҚ·й”®

е°Ҷд»ҘдёҠзј©ж”ҫ+ж— й—ӘзғҒзҡ„flameshotеҗҜеҠЁе‘Ҫд»ӨеҠ еҲ°еҝ«жҚ·й”®пјҡ

settings-> keyboard -> custom shortcuts -> ж·»еҠ commandе’Ңshortcut

иҝңзЁӢдёҺе…ұдә«

UbuntuиҝңзЁӢзҷ»еҪ•еҲ°Windowsдё»жңә

жҺЁиҚҗremminaпјҢеҗҢж—¶ж”ҜжҢҒRDPе’ҢVNCеҚҸи®®пјҡsudo apt install remmina

й»ҳи®Өи®ҫзҪ®йңҖиҰҒж”№пјҡд»Ҙзӣ®ж Үдё»жңәзҡ„еҲҶиҫЁзҺҮжҳҫзӨә; зӣ‘еҗ¬зӣ®ж Үдё»жңәзҡ„й”®зӣҳеҝ«жҚ·й”®пјҢеңЁзӣ®ж Үдё»жңәеҶ…зҡ„й”®зӣҳйғҪжҳҜеҜ№иҜҘдё»жңәзҡ„ж“ҚдҪң

Ubuntuи®ҝй—®Windowsе…ұдә«и·Ҝеҫ„

win10е…ұдә«жҹҗдёӘеҲҶеҢәзӣҳпјҡеҸій”®еҲҶеҢә->й«ҳзә§е…ұдә«->жү“ејҖиҜ»еҶҷи®ҝй—®пјӣи®°еҪ•ipconfigзҡ„IP

Ubuntuе·Із»Ҹе®үиЈ…sambaеҗҺпјҢеңЁж–Үд»¶з®ЎзҗҶ->е…¶д»–дҪҚзҪ®->иҝһжҺҘеҲ°жңҚеҠЎеҷЁ, иҫ“е…Ҙwin10дё»жңәIPпјҢдјҡиҫ“е…Ҙwin10дё»жңәзҡ„з”ЁжҲ·еҗҚпјҢеҜҶз ҒпјҢеҹҹеҗҚеҗҺеҸҜд»ҘиҝһжҺҘпјӣеҸӘиғҪжү“ејҖwin10е·Іе…ұдә«зҡ„еҲҶеҢәзӣҳгҖӮ

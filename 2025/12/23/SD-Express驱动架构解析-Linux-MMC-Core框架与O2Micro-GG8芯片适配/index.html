SD Express是SD协会发布的第七代SD标准，核心特性：

最大速度 ~100 MB/s (UHS-II) ~4 GB/s (PCIe 4.0 x2)
总线协议 SD/UHS-II PCIe/NVMe

总结

本文分析了Linux MMC子系统中SD Express的支持框架：

  1. MMC Core提供的能力：

    • Capability声明机制
    • 状态机管理
    • 回调接口定义

  2. GG8芯片的适配点：

    • Device ID注册
    • init_sd_express回调实现
    • PCIe模式切换控制
    • 错误恢复机制

  3. 协作流程：

    • MMC Core主导流程
    • Host驱动提供硬件操作
    • NVMe驱动负责接管

这种分层架构设计使得不同厂商的SD Express控制器可以复用通用的MMC Core代码，只需实现厂商特定的硬件操作。
